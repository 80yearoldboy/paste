Supabase 配置

修复邮箱验证链接失效问题

步骤 1: 在Supabase中配置正确的网站URL

登录Supabase控制台 → Authentication → URL Configuration

设置 Site URL 为你的GitHub Pages地址:

https://yourusername.github.io/your-repo
步骤 2: 添加重定向URL

在 Redirect URLs 中添加:

https://yourusername.github.io/your-repo/**
步骤 3: 暂时禁用邮箱验证 (可选)

在 Authentication → Settings → Auth Settings 中:

将 Confirm email 设置为 "Off" 进行测试

故障排除

  • 确保 Supabase URL 和 Key 正确无误
  • 检查网站URL配置是否正确
  • 验证链接通常在24小时后过期
  • 尝试重新发送验证邮件
